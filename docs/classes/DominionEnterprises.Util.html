Ensures that $valueToEnsure is equal to $valueToCheck or it throws

ensure(mixed $valueToEnsure, mixed $valueToCheck, null|string|\Exception $exception = null, array|null $exceptionArgs = null) : mixed
static

Can be used like: $result = ensure(true, is_string('boo')) Or like: $result = ensure(true, is_string('boo'), 'the message') Or like: $result = ensure(true, is_string('boo'), 'MyException', array('the message', 2)) Or like: $result = ensure(true, is_string('boo'), new MyException('the message', 2))

throws

if $valueToEnsure !== $valueToCheck

if $exception was not null, a string, or an Exception

Arguments

$valueToEnsure

mixed

the value to throw on if $valueToCheck equals it

$valueToCheck

mixed

the value to check against $valueToEnsure

$exception

null|string|\Exception

null, a fully qualified exception class name, string for an Exception message, or an Exception.

The fully qualified exception class name could also be an alias in getExceptionAliases()

$exceptionArgs

array|null

arguments to pass to a new instance of $exception. If using this parameter make sure these arguments

match the constructor for an exception of type $exception.

Response

mixed

returns $valueToCheck
