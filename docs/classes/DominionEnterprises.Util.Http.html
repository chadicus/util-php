Static class with various HTTP related functions.

Parses HTTP headers into an associative array.

parseHeaders(string $rawHeaders) : array
static

Example: $headers = "HTTP/1.1 200 OK\r\n".

       "content-type: text/html; charset=UTF-8\r\n".
       "Server: Funky/1.0\r\n".
       "Set-Cookie: foo=bar\r\n".
       "Set-Cookie: baz=quux\r\n".
       "Folds: are\r\n\treformatted\r\n";

print_r(\DominionEnterprises\HttpUtil::parseHeaders($headers)); The above example will output:

Array
(
    [Response Code] => 200
    [Response Status] => OK
    [Content-Type] => text/html; charset=UTF-8
    [Server] => Funky/1.0
    [Set-Cookie] => Array
    (
      [0] => foo=bar
      [1] => baz=quux
    )
    [Folds] => are reformatted
)
throws

Thrown if unable to parse the headers

Arguments

$rawHeaders

string

string containing HTTP headers

Response

array

the parsed headers

Generate URL-encoded query string

buildQueryString(array $parameters) : string
static

Example: $parameters = array( 'param1' => array('value', 'another value'), 'param2' => 'a value', );

$queryString = \DominionEnterprises\HttpUtil::buildQueryString($parameters);

echo $queryString

Output:

param1=value¶m1=another+value¶m2=a+value

Arguments

$parameters

array

An associative array containing parameter key/value(s)

Response

string

the built query string

Get an array of all url parameters.

getQueryParams(string $url) : array
static
throws

if $url was not a string

Arguments

$url

string

The url to parse such as http://foo.com/bar/?id=boo&another=wee&another=boo

Response

array

such as ['id' => ['boo'], 'another' => ['wee', 'boo']]

getQueryParamsCollapsed(string $url, array $expectedArrayParams = array()) : array
static
throws

if $url was not a string

if a parameter is given as array but not included in the expected array argument

Arguments

$url

string

The url to parse such as http://foo.com/bar/?single=boo&multi=wee&multi=boo

$expectedArrayParams

array

List of parameter names which are not collapsed.

Response

array

such as ['single' => 'boo', 'multi' => ['wee', 'boo']] if 'multi' is given in $expectedArrayParams
